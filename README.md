# CSI-Book-Analyzer

Create your own personal digital library using links and titles from gutenberg.org! First download both the `main.py` and `EN-Stopwords.txt` files and access them to get started.


# To Begin:

  1. Open the main.py file
  2. With user input, 1-5 demonstrates (in order) the following:
       - Addition of a book to the library
       - Removal of a book from the library
       - Ability to update a book's url
       - Ability to analyze the top 10 most common words of a book
  3. First add a book by pressing 1 and entering the book's title, then copy and paste a .txt url from the book of choice (books and links found on gutenberg.org)
  4. Enjoy your python-coded library!
